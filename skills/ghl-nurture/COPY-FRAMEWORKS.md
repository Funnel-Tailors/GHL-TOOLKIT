# Copy Frameworks — Frameworks de Copywriting para Nurturing

## Framework PAS (Problem → Agitate → Solve)
Para emails de nurturing donde necesitas generar urgencia.

```
PROBLEM: "¿Sigues [describir el problema]?"
AGITATE: "Cada día que pasa, [consecuencia del problema]. [Nombre caso] estaba igual hasta que..."
SOLVE: "En [tiempo], consiguió [resultado]. Tu llamada es el primer paso: {{calendar_link}}"
```

## Framework BAB (Before → After → Bridge)
Para social proof y casos de éxito.

```
BEFORE: "[Nombre] estaba [situación antes]. [Dolor específico]."
AFTER: "Hoy [resultado concreto]. [Métrica impresionante]."
BRIDGE: "¿Cómo? En una llamada de 30 min diseñamos el plan. Agenda: {{calendar_link}}"
```

## Framework 4Ps (Promise → Picture → Proof → Push)
Para mensajes de cierre o última oportunidad.

```
PROMISE: "Puedo ayudarte a conseguir [resultado específico]."
PICTURE: "Imagina [descripción vivida del resultado]."
PROOF: "[X] clientes ya lo han conseguido. Media de [métrica]."
PUSH: "Últimas [X] plazas. Agenda antes del [fecha]: {{calendar_link}}"
```

## Reglas de SMS (160 chars max)

1. Nombre del contacto SIEMPRE al inicio
2. Una sola idea por mensaje
3. Un solo CTA (link o acción)
4. Tono conversacional, como un mensaje real
5. NO usar mayúsculas para gritar
6. NO emojis excesivos (max 1)

## Reglas de Email

1. Subject max 50 chars, personalizado con nombre
2. Preview text que genere curiosidad
3. Primer párrafo: hook directo (max 2 líneas)
4. Body: max 200 palabras
5. Un solo CTA principal (puede repetirse 2 veces max)
6. P.D.: urgencia o bonus adicional
7. Mobile-first: párrafos cortos, sin imágenes pesadas
