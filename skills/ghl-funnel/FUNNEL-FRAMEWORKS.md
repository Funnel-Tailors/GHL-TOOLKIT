# Funnel Frameworks — Arquetipos de Embudos

## 1. Application Funnel (High Ticket)

**Ideal para**: Consultoría, coaching, servicios premium (>2000€)

```
Landing Page (VSL o long-form)
    ↓
Multi-Step Application Form (5-8 preguntas)
    ↓ scoring automático
Thank You Page (personalizada por score)
    ↓
[HOT] → Calendario embebido → Confirmación → Llamada
[WARM] → Nurturing 5-7 días → Calendario → Llamada
[COLD] → Contenido educativo → Recualificación eventual
```

**Preguntas típicas**:
1. ¿En qué sector estás? (ICP fit)
2. ¿Cuál es tu facturación actual? (capacidad)
3. ¿Qué problema quieres resolver? (dolor)
4. ¿Cuándo necesitas resolverlo? (urgencia)
5. ¿Has invertido en soluciones antes? (madurez)
6. ¿Cuánto puedes invertir? (presupuesto)
7. ¿Eres el decisor? (autoridad)

**Clave**: La propia aplicación genera sensación de exclusividad. "No aceptamos a todo el mundo."

## 2. VSL Funnel (Mid Ticket)

**Ideal para**: Servicios, infoproductos (500-2000€)

```
Ad → Landing con Video (VSL 15-45 min)
    ↓
Form corto bajo el video (3-4 preguntas)
    ↓ scoring
Thank You Page con CTA
    ↓
[Vio >50% video + form] → Calendario directo
[Vio <50% video + form] → Nurturing + replay del video
[Solo form, sin video] → Nurturing educativo
```

**Clave**: El video hace el trabajo pesado de venta. El form cualifica. La llamada cierra.

## 3. Webinar Funnel

**Ideal para**: SaaS, cursos, membresías

```
Ad → Landing de registro
    ↓
Form de registro (2-3 preguntas + scoring)
    ↓
Página de confirmación + upsell inmediato
    ↓
Secuencia pre-webinar (3 emails + 2 SMS)
    ↓
Webinar (live o evergreen)
    ↓
Oferta en el webinar → Checkout
    ↓ si no compra
Secuencia post-webinar (replay + urgencia + bonus)
    ↓
[No compra en 5 días] → Llamada de cierre (solo para leads cualificados)
```

**Scoring del webinar**:
- Se registró: +5
- Abrió emails pre-webinar: +5
- Asistió al webinar: +15
- Se quedó > 50%: +10
- Clickeó link de oferta: +10
- Visitó checkout: +5

## 4. Quiz Funnel

**Ideal para**: Lead generation, segmentación, cualquier ticket

```
Ad → Landing con Quiz
    ↓
Quiz interactivo (7-12 preguntas con progress bar)
    ↓ scoring + segmentación
Página de resultados personalizados
    ↓
[Resultado A] → CTA específico para segmento A
[Resultado B] → CTA específico para segmento B
[Resultado C] → CTA específico para segmento C
    ↓
Form de contacto (email + phone para recibir resultados)
    ↓
Nurturing personalizado por segmento
```

**Clave**: El quiz genera engagement altísimo (completions >70%). Los resultados personalizados demuestran autoridad. Cada segmento recibe nurturing adaptado.

## 5. Challenge Funnel

**Ideal para**: Comunidad, lanzamientos, engagement largo

```
Ad → Landing de Challenge (5-7 días)
    ↓
Registro (email + phone + 2-3 preguntas)
    ↓
Día 1-5: Emails/SMS diarios con tarea + contenido
    ↓ scoring de participación
[Completó tareas + activo] → Oferta + Calendario
[Parcialmente activo] → Nurturing con highlights
[Inactivo] → Re-engagement o drop
```

**Scoring del challenge**:
- Cada tarea completada: +10
- Cada email abierto: +3
- Cada tarea compartida en redes: +5
- Pregunta en el grupo: +5
- Asistió a sesión en vivo: +10

## Branching Dinámico

Todos los funnels comparten un concepto: **branching basado en score**.

```
Score calculado
    │
    ├─ ≥ 70 (HOT)
    │   ├─ Thank you page A (urgencia, calendario directo)
    │   ├─ SMS inmediato (personalizado, CTA agenda)
    │   └─ Recordatorios directos
    │
    ├─ 40-69 (WARM)
    │   ├─ Thank you page B (social proof, valor)
    │   ├─ Email con caso de estudio
    │   └─ Nurturing educativo → agenda
    │
    └─ < 40 (COLD)
        ├─ Thank you page C (contenido gratuito)
        ├─ Email educativo (no venta)
        └─ Monitorizar engagement → recualificar si sube
```
